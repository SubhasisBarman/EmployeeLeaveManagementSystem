Step 1: Download the file from Google Drive.

Step 2: Unzip the file using 7-Zip or any extraction tool.

Step 3: In the Database folder, there is a .bak file. Restore this .bak file in Microsoft SQL Server.

Step 4: Open the project in Visual Studio by 'DotNetAssignmentMVC.sln' file.

Step 5: Generate the Connection String and replace it in the 'DefaultConnection' section of the appsettings.json file.

Step 6: Build and run the project.
